# 微信小程序开发辅助决策助手

## 概述

本技能为微信小程序开发提供专业的多维度可行性决策分析。用户只需提供产品名称和方向，系统自动完成市场调研、竞品分析、趋势评估，生成专业的可视化HTML决策报告。

## 核心能力

1. **微信指数趋势分析** - 通过搜索获取关键词在微信生态的热度趋势
2. **竞品数量与格局分析** - 搜索同类小程序数量、头部竞品信息
3. **行业规模与趋势** - 获取最新行业报告数据、市场规模、增长率
4. **流量潜力评估** - 分析搜索量、自然流量机会、微信搜一搜SEO潜力
5. **商业模式建议** - 基于品类特征推荐最优变现模式
6. **推广策略建议** - 裂变、SEO、私域、广告等推广路径
7. **开发技巧与避坑** - 技术选型、审核要点、成本估算
8. **综合可行性评分** - 多维度加权评分，给出明确决策建议

## 使用方式

```
触发词 + 产品名称 + 方向描述
```

示例：
- "小程序决策：做一个钓鱼技巧交流社区小程序"
- "帮我评估做AI写作小程序可行吗"
- "分析宠物社交小程序的可行性"

## 工作流程

### 第一阶段：数据采集（并行搜索）

使用 WebSearch 工具并行搜索以下维度：

1. **微信指数**：搜索 `{产品关键词} 微信指数 热度 趋势`
2. **竞品分析**：搜索 `微信小程序 {产品方向} 竞品 排行榜 数量`
3. **行业数据**：搜索 `{产品方向} 微信小程序 市场规模 2025 2026`
4. **商业模式**：搜索 `{产品方向} 小程序 变现 盈利模式`
5. **推广方法**：搜索 `{产品方向} 小程序 推广 裂变 引流`

### 第二阶段：深度分析（可选WebFetch）

对关键搜索结果中数据丰富的页面进行 WebFetch 获取详细信息。

### 第三阶段：报告生成

运行报告生成脚本生成HTML报告：

```bash
python {baseDir}/scripts/report_generator.py \
  --name "产品名称" \
  --direction "产品方向" \
  --output "output_path.html" \
  --wechat-index '{"score": 75, "trend": "上升", "detail": "..."}' \
  --competitors '{"count": 45, "top3": [...], "detail": "..."}' \
  --industry '{"market_size": "...", "growth": "...", "detail": "..."}' \
  --business-model '{"recommend": [...], "detail": "..."}' \
  --promotion '{"strategies": [...], "detail": "..."}' \
  --development '{"tech_stack": "...", "cost": "...", "tips": [...], "detail": "..."}'
```

或者直接在Python中调用：

```python
import json, sys
sys.path.insert(0, '{baseDir}/scripts')
from report_generator import generate_report

data = {
    "name": "产品名称",
    "direction": "产品方向",
    "wechat_index": {...},
    "competitors": {...},
    "industry": {...},
    "business_model": {...},
    "promotion": {...},
    "development": {...}
}

html = generate_report(data)
with open('output.html', 'w', encoding='utf-8') as f:
    f.write(html)
```

## 报告结构

生成的HTML报告包含以下章节：

1. **封面页** - 产品名称、评估时间、总体评分
2. **执行摘要** - 核心结论与建议（可行/谨慎/暂缓）
3. **微信指数分析** - 搜索热度趋势、相关关键词
4. **竞品格局分析** - 竞品数量、头部玩家、市场饱和度
5. **行业市场分析** - 市场规模、增长率、政策环境
6. **流量潜力评估** - 搜索流量、自然增长、SEO机会
7. **商业模式建议** - 推荐变现模式、收入预估
8. **推广策略** - 冷启动方案、裂变设计、投放建议
9. **开发指南** - 技术选型、成本估算、审核避坑
10. **风险提示** - 政策风险、竞争风险、技术风险
11. **综合评分与决策** - 加权评分矩阵、最终建议

## 评分模型

基于以下6个维度加权评分（满分100）：

| 维度 | 权重 | 评估标准 |
|------|------|----------|
| 微信热度 | 25% | 微信指数趋势、搜索量级 |
| 竞争格局 | 20% | 竞品数量、头部垄断程度 |
| 市场前景 | 20% | 行业规模、增长率 |
| 变现能力 | 15% | 商业模式清晰度、ARPU潜力 |
| 开发可行性 | 10% | 技术难度、成本、审核风险 |
| 流量获取 | 10% | 自然流量、SEO、裂变潜力 |

评分等级：
- ≥80分：✅ **强烈建议做** — 市场机会明确，竞争适中
- 65-79分：🟡 **谨慎推进** — 有机会但需差异化策略
- 50-64分：⚠️ **暂缓观望** — 需先验证核心假设
- <50分：❌ **不建议做** — 市场风险过高

## 注意事项

1. 所有数据基于公开信息搜索，时效性可能有限
2. 微信指数为估算值，非官方精确数据
3. 竞品数量基于搜索结果统计，可能有遗漏
4. 最终决策需结合团队实际资源与能力判断
