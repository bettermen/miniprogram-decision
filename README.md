# 📱 微信小程序开发辅助决策助手

> 输入产品名称和方向，AI 自动完成市场调研、竞品分析、趋势评估，生成专业可视化决策报告。

## ✨ 核心能力

| 维度 | 分析内容 |
|------|----------|
| 🔍 微信指数 | 搜索热度趋势、关联热词 |
| ⚔️ 竞品格局 | 同类数量、头部玩家、饱和度 |
| 📈 行业市场 | 规模、增长率、政策环境 |
| 🚀 流量潜力 | 搜一搜SEO、裂变、自然流量 |
| 💰 商业模式 | 推荐变现模式、收入潜力 |
| 📢 推广策略 | 冷启动→增长→规模化路径 |
| 🔧 开发指南 | 技术栈、成本、避坑清单 |
| ⚠️ 风险提示 | 合规/竞争/技术风险矩阵 |
| 🎯 综合评分 | 6维加权评分+明确决策建议 |

## 📊 评分模型

| 维度 | 权重 |
|------|------|
| 微信热度 | 25% |
| 竞争格局 | 20% |
| 市场前景 | 20% |
| 变现能力 | 15% |
| 开发可行性 | 10% |
| 流量获取 | 10% |

- ≥80 → ✅ 强烈建议做
- 65-79 → 🟡 谨慎推进
- 50-64 → ⚠️ 暂缓观望
- <50 → ❌ 不建议做

## 🚀 使用方式

```
小程序决策：做一个XX小程序
```

系统自动并行搜索5大维度数据，生成完整HTML决策报告。

## 📦 安装

作为 WorkBuddy/ClawHub 技能安装：

```bash
# 克隆到技能目录
git clone https://github.com/bettermen/miniprogram-decision.git ~/.workbuddy/skills/miniprogram-decision/
```
